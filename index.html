BlackJack

Nylig laget jeg en blackjack-nettside ved hjelp av HTML, CSS og JavaScript. HTML ga struktur til spillbrettet, CSS sørget for estetikk, og JavaScript styrte spilllogikken, inkludert kortfordeling og vinneravgjørelser. Resultatet er en interaktiv og engasjerende blackjack-opplevelse som kombinerer både design og funksjonalitet

Roulette

Nylig utviklet jeg en spennende roulette-nettside ved hjelp av HTML, CSS og JavaScript. HTML ble brukt til å strukturere layouten, CSS ga en visuelt tiltalende design, og JavaScript tok seg av spilllogikken. Jeg implementerte funksjonalitet for å snurre ruletthjulet, vise resultatene og gi en spennende spillopplevelse. Prosjektet ga meg ikke bare verdifull erfaring innen webutvikling, men demonstrerte også kraften i å kombinere ulike teknologier for å skape en engasjerende og interaktiv rouletteopplevelse.

Bitcoin Exchange

Gjennom utviklingen av mitt bitcoin-børsnettsted med HTML, CSS og JavaScript, har jeg fokusert på å styrke mine front-end og back-end ferdigheter. HTML og CSS ble anvendt for å skape en brukervennlig grensesnitt, mens JavaScript-objekter ble implementert i back-end for å håndtere datastrukturer og forbedre nettstedets funksjonalitet. Dette prosjektet har vært en verdifull læringsopplevelse som har utvidet min forståelse av webutvikling.
